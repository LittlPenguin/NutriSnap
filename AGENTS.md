# NutriSnap 开发约束

本文件约束 `end/` Git 仓库内的后续开发。修改源码、测试、配置或说明前，应先阅读本文件。

## 项目定位

- 项目名称：基于 Streamlit 与迁移学习的食物图像识别与热量估算系统。
- 展示名称：NutriSnap 轻食记录。
- 系统形态：Streamlit Web App，兼容 PC 浏览器和手机浏览器，不开发原生 APK。
- 课程目标：完成可运行演示、源码、测试和课程报告所需截图。

## 固定技术栈

- Python
- Streamlit
- PyTorch
- ResNet18 迁移学习
- Food-101 子集
- SQLite
- Pillow / OpenCV
- Pandas / NumPy
- OpenAI GPT-5 API

除非用户明确重新变更方案，不使用 Flask、React、shadcn、原生 Android / iOS App 替代当前方案。

## 功能边界

- 食物识别由 PyTorch ResNet18 模型完成。
- GPT-5 只负责根据食物名称、重量、估算热量和用户目标生成饮食建议，不作为图像识别模型。
- GPT-5 调用失败、API Key 未配置或超时时，必须返回本地规则建议。
- 热量估算必须基于“识别类别 + 用户输入重量或默认重量 + 每 100g 热量表”。
- 禁止写成仅凭图片自动精确估重或精准计算热量。
- 所有热量文案必须使用“估算热量”“仅供饮食记录参考”等表达。
- 系统不作为医学或营养诊断，不提供治疗建议。

## UI 约束

- UI 以 Open Design 最新稿为基准：轻食记录风，绿色主色，柔和橙色强调。
- Streamlit 页面使用五个主页面：食物识别、历史记录、热量表、统计分析、系统说明。
- PC 端必须呈现顶部导航；移动端必须呈现底部导航视觉。
- UI 开发必须尽可能还原 `opendesign-nutrisnap/index.html` 中的移动端 5 页和 PC 端 5 页结构、色彩、卡片、进度条、指标卡和边界说明。
- 系统说明页不展示技术栈，只展示项目简介、使用步骤、估算限制和免责声明。
- 关键示例统一为：披萨，置信度 93.0%，重量 150g，每 100g 热量 266 kcal / 100g，估算热量 399 kcal。

## 阶段开发与提交

- 按阶段开发，每完成一个阶段必须进行一次 Git 提交。
- 每个阶段提交前必须更新 `doc/10-阶段开发计划.md` 中对应阶段的 `已完成`、`完成提交`、`验证证据`、`阻塞记录` 字段。
- 提交信息必须使用中文，并写清楚“较上次修改了什么”。
- `doc/*.md` 属于项目文档，需要纳入 Git；课程任务书和参考模板等 Word 文件不提交。
- `opendesign-nutrisnap/` 只作为本地设计归档，不提交。

## 阻塞记录

- 遇到用户侧事项时，不应阻塞其他可开发阶段，应记录到 `doc/开发阻塞记录.md` 后继续。
- 用户侧事项包括但不限于：真实 `OPENAI_API_KEY`、Food-101 原始数据、训练后的 `models/food_resnet18.pth`、真实移动端设备验收、课程视频录制素材。
- 记录阻塞项时应写明影响范围、当前降级方案和后续处理动作。

## 密钥与数据

- 不得提交真实 API Key。
- 使用 `.env` 保存 `OPENAI_API_KEY` 和 `OPENAI_MODEL`。
- `.env`、SQLite 数据库、模型权重、数据集、训练输出不提交。
- `.env.example` 可以提交，只包含变量名和示例配置。

## 开发与验证

- 新行为优先写测试，再实现。
- 核心服务必须能在没有模型权重和没有 API Key 的情况下可测试、可演示。
- 推送前至少运行 `pytest` 和 `ruff check .`。
- 启动 Streamlit 进行验证后，应停止本地服务。

## 当前修正规则

- OpenAI 配置必须从 `.env` 读取，并支持 `OPENAI_API_KEY`、`OPENAI_BASE_URL`、`OPENAI_MODEL`。
- 不得在源码、文档、测试输出或提交信息中暴露真实 API Key。
- Streamlit 页面切换必须使用站内状态切换，禁止把主导航实现为普通 `<a href="?page=...">` 链接跳转。
- 全局样式必须为 Streamlit 默认头部预留安全间距，避免遮挡 NutriSnap 顶部品牌栏。
- Food-101 数据准备脚本必须支持下载、整理和验收报告；`dataset/` 仍然不提交。
- 未训练并放入 `models/food_resnet18.pth` 前，不得宣称已完成真实 ResNet18 推理验收。
